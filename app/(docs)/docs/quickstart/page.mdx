# Quickstart

## 1. Connect your wallet

Open the [app](/research) and connect a wallet. Pyxis uses **Sign-In With
Ethereum (SIWE)** — you sign a message to prove ownership. No transaction, no
gas. The signature unlocks your private research history.

## 2. Ask a question

Type any token, chain, protocol, or narrative — for example a ticker, a contract
address, or a theme like "restaking". The agents run in parallel and a briefing
arrives in roughly 30–40 seconds.

## 3. Read the briefing

The briefing is organized into sections chosen for your topic, and closes with a
**Data Freshness** table. Live data is marked `[live]`; recently cached data is
marked `[cached]`.

## 4. Share it

Every briefing has a public, read-only link (`/b/<id>`). Anyone with the link can
read the briefing — your wallet address is never exposed.

<Callout>
Pyxis is **free during beta.** You'll see the original pricing struck through in
the app; nothing is charged today.
